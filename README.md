# Signal Analysis with FFT — Scientific Initiation Research (PIBIC)

Python notebooks developed during an undergraduate research fellowship (PIBIC) focused on signal processing and dimensionality reduction applied to real measurement data.

## Topics covered:

1. Fast Fourier Transform (FFT) and Inverse FFT applied to signals from a Pinnacle Plus device across multiple frequencies (10–200 kHz)
2. Approximate FFT reconstruction and frequency-domain filtering
3. Truncated Singular Value Decomposition (SVD) for image compression
4. Linear and multilinear regression experiments
5. PCA (Principal Component Analysis) on tabular datasets

**Stack**: Python · NumPy · SciPy · Matplotlib · Jupyter Notebook

**Period**: 2021–2022
